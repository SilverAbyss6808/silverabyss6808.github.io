Neko-Sakuba


      Neko-Sakuba is my current passion project! This game is a 2D top-down farming simulator/RPG that follows the player (a cat!) as they wake up in an unfamiliar land called Nekita. They must venture through the world to try to find out what happened and how to get home. Neko-Sakuba is currently in the beginning stages of development in Godot Engine, which I've found very fun to work in. Though I'm still new to game development, I'm learning a lot and excited to both undertake this journey and finish with a complete game that I can be proud of!
